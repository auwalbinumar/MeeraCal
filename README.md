 MeeraCal Application Overview
MeeraCal (also referred to as MeraCal) is a specialized desktop application developed by **Elhasub Soft_Solution** and designed by **A.B. Umar**. The software provides automated computation and records management for the calculation and distribution of inheritance shares to legitimate heirs following Islamic law.

 Key Capabilities & Jurisprudence
The core purpose of MeeraCal is to collect necessary data regarding a deceased individual's estate and family structure, then compute precise inheritance shares. To accommodate varying legal interpretations, the system is engineered to calculate distribution options in strict accordance with the **four major Islamic jurisprudence schools (Mazhabs):
* Hanafi
* Maliki
* Shafi’i
* Hanbali

The application accurately processes complex legal inheritance scenarios, including allocations for parents, grandparents, children, grandchildren, spouses, siblings, nieces/nephews, and uncles. It natively handles special legal conditions, such as the **'RAD' case**, which involves redistributing remaining estate fractions back to assigned rate-holders.

 Interface & Operational Features
MeeraCal features a localized user interface that supports three languages: **English, Arabic, and Hausa**. It includes distinct menu options categorized for data entry, system updates, and navigation:

The File Menu: Houses application settings, developer information, activation keys, and language localization configurations.


The Calculation System: Divided into an *Add Page* for initial data entry (processing the deceased's name, total financial assets, currency types, and active heirs) and an *Edit Form* for modifying, duplicate copying, or deleting stored entries from the application database.


The View System: Compiles all saved individual or aggregate profiles into a structured, tabular reporting interface that showcases calculations, legal rates, and *Ta’asib* assignments.


To facilitate use in legal registries or institutional settings, the application features an administrative user-access framework to ensure only authorized personnel handle data. Computed inheritance breakdowns can also be directly exported to soft copies (Word or PDF formats) or routed to external hardware for physical printing.
